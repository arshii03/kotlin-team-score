# Kotlin Team Score Counter App

This is a simple Android application built in Kotlin to manage scores for two teams, **Team A** and **Team B**. Users can increment the scores by 1, 2, or 3 points for each team and view the updated scores in real-time.

## Features
- Increment scores for Team A and Team B by 1, 2, or 3 points.
- Display updated scores for both teams.
- Clean and simple UI built with ConstraintLayout.
- Modular Kotlin code for score management.


## Prerequisites
Before running this application, ensure you have the following installed:
- Android Studio (latest version recommended)
- Kotlin support enabled in Android Studio
- Android SDK version 21 or above
